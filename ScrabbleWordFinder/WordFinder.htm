Scrabble Word Finder Example

This example uses a provided list of words as its dictionary. It takes the letters provided (max 7) and finds the possible words from the dictionary. It then computes the Scrabble value of the word and sorts the words from greatest value to least.

To help simulate a real-world example, this application will also cache any values it calculates for a word to reduce repeative processing.
